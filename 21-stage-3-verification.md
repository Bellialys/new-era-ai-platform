# Что реализовано в UI

## Форма

- Поле ввода prompt.
- Выбор моделей через checkbox.
- Счётчик выбранных моделей.
- Кнопка запуска сравнения.
- Кнопка очистки.

## Валидация

- Prompt должен быть минимум 3 символа.
- Prompt ограничен максимумом 8000 символов.
- Нужно выбрать минимум 2 модели.
- В MVP можно выбрать максимум 3 модели.
- Для лимита моделей используется единый текст ошибки.
- Ошибка показывается в интерфейсе.

## Состояния

- Empty state - ответов ещё нет.
- Loading state - имитация генерации ответов.
- Error state - ошибка валидации.
- Success state - карточки mock-ответов.
- Старые responses сбрасываются при изменении prompt или выбора моделей.

## Ответы моделей

- Ответы появляются после нажатия кнопки.
- У каждого ответа есть модель, роль в UI, статус и latency.
- Текст ответа берётся из `answerText`.
- При ошибке модели карточка может показать `errorMessage` и `errorCode`.
- Успешный ответ можно выбрать лучшим.
- Ошибочный ответ нельзя выбрать победителем.
- Победитель получает визуальную метку `Winner`.

# Ручной тест в браузере

После запуска `npm run dev` открыть:

```text
http://localhost:3000/arena
# страница Prompt Arena
```

Проверить сценарии:

1. Открыть `/arena`.
2. Нажать запуск с пустым prompt.
3. Убедиться, что появляется ошибка.
4. Ввести задачу меньше 3 символов.
5. Убедиться, что появляется ошибка.
6. Проверить, что prompt ограничивается 8000 символами.
7. Оставить выбранной только одну модель.
8. Убедиться, что появляется ошибка.
9. Попробовать выбрать больше 3 моделей.
10. Убедиться, что появляется единое сообщение о лимите моделей.
11. Ввести нормальный prompt.
12. Выбрать 2 или 3 модели.
13. Нажать запуск сравнения.
14. Убедиться, что появляется loading state.
15. Убедиться, что после загрузки появляются mock-ответы.
16. Изменить prompt после появления ответов.
17. Убедиться, что старые ответы исчезают.
18. Запустить сравнение снова.
19. Нажать `Выбрать лучшим`.
20. Убедиться, что выбранная карточка получает метку `Winner`.
21. Нажать `Очистить`.
22. Убедиться, что форма и ответы сбрасываются.
