# Privacy and Data Retention — New Era AI Platform v1.1

## Какие данные мы собираем

| Данные | Где хранятся | Зачем |
|---|---|---|
| Email, хэш пароля | Supabase Auth | Аутентификация |
| Тексты промптов | Supabase `tasks` | История сравнений |
| Ответы AI-моделей | Supabase `model_responses` | История сравнений |
| Голоса (winner votes) | Supabase `votes` | Персональная аналитика |
| Guest session ID | httpOnly cookie `na_guest` | Анонимная сессия |
| Anonymous session ID | Supabase `tasks` | Привязка истории к гостю |

## Что мы НЕ храним

- IP-адреса пользователей (не сохраняются намеренно)
- Prompt тексты в обычных server logs (только в Supabase)
- Платёжные данные (нет платёжной системы на v1.1)
- Данные OAuth / social login (не используется)

## Срок хранения данных (политика v1.1)

| Тип данных | Срок хранения |
|---|---|
| Prompt history (аутентифицированный пользователь) | Бессрочно (пока активен аккаунт) |
| Prompt history (гость) | 90 дней с момента последней активности |
| Guest cookie | 30 дней (expires) |
| Аккаунт пользователя | До удаления аккаунта |

> **Примечание:** Автоматическое удаление гостевых данных через 90 дней не реализовано на v1.1.
> Требует Supabase scheduled function или cron job. Запланировано для v1.2.

## Передача данных третьим лицам

| Получатель | Данные | Основание |
|---|---|---|
| OpenRouter | Текст промпта | Необходимо для работы AI-сравнений |
| Vercel | Логи запросов (без prompt body) | Хостинг платформы |
| Supabase | Все данные выше | База данных платформы |

Prompt тексты передаются в OpenRouter для получения AI-ответов. OpenRouter policy: https://openrouter.ai/privacy
